The Database is Out of Date

It needs to be updated before uBiblio can be used. This process is automatic, and can be done by any Administrator by clicking here. Before you begin, we recommend backing up the database.

For instructions on how to backup the database, please see the readme file included in this application or visit the GitHub page.

After this update, you will be able to manage backups in a simpler manner inside the application (by visiting the new admin page via the menu).
